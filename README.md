Folders:
app: Contains all main server-side code
    server-side-game: Handles most base game logic.
        card.js: Creates cards that can be used for amalgamations.
        amalgamation.js: Handles amalgamations on the field.
        cardDeck.js: List Wrapper containing a List of cards.
        cardHand.js: Map Wrapper containing a Map of cards.
        player.js: Handles player card and amalgamation data. Provides functions to perform player actions.
public: Contains all main public game code.
test.js: Used to run sample games and test code.
